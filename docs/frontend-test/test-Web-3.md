## 4. 缺陷修复情况

根据前两轮测试中找到的缺陷，针对性地进行缺陷修复情况考察。

### 4.1 第一轮测试报告缺陷修复情况

第一轮测试发现的 bug 如下：

1. 若用户名中包含下划线，则用户无法通过“用户名 + 密码”的方式正常登录。
2. 注册环节中，对邮箱和密码的字符范围限制不足：用户可以在邮箱和密码中包含非法字符（例如中文字符）。
3. 若用户在邮箱中包含非法字符，则注册后用户无法正常登录。

接下来，对三个缺陷的修复状况分别进行考察。

#### (1) 用户名中包含下划线的登录问题
